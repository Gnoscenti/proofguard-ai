# ProofGuard AI: The Deterministic Governance Layer for Agentic Systems

ProofGuard AI is the industry-standard Trust Plane for AI. It provides a deterministic governance layer ensuring agents operate within strict boundaries.

## Key Features
- Hallucination Prevention
- Audit-Ready Compliance (IMDA/AICM 243)
- Machine-Readable Rules
- Fail-Closed Security

## Architecture
Built with Atlas, signed JWTs, and immutable ledgers. Stack: TypeScript, React, Node.js, Drizzle.

## Installation
pnpm install && pnpm run dev

## Compliance
SOC2, FIPS.
